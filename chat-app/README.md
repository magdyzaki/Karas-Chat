# تطبيق الشات

شات ويب + موبايل: محادثات فردية ومجموعات، رسائل نصية وصور/ملفات، مع تسجيل دخول.

## التشغيل المحلي

### الباكند (المنفذ 5000)
```bash
cd chat-app/backend
npm install
npm start
```

### الواجهة (مع الـ proxy للسيرفر المحلي)
```bash
cd chat-app/frontend
npm install
npm run dev
```

ثم افتح المتصفح على الرابط الذي يظهر (مثلاً http://localhost:5173). لا حاجة لضبط `VITE_API_URL` في التطوير؛ الـ proxy يوجّه `/api` و`/socket.io` إلى الباكند.

## المتطلبات
- Node.js
- متصفح حديث (ويب أو موبايل)

## المميزات
- تسجيل دخول / إنشاء حساب
- محادثة فردية بين أي مستخدمين
- إنشاء مجموعة واختيار الأعضاء
- رسائل نصية فورية (Socket.io)
- إرسال صور وملفات (رفع إلى السيرفر وعرض في الشات)
- واجهة متجاوبة (ويب وموبايل)

## النشر
- **الباكند:** ارفعه على أي استضافة Node (مثل Koyeb، Render) وضبط `PORT` و`JWT_SECRET` و`CORS_ORIGIN`.
- **الواجهة:** ابنِها بـ `npm run build` وارفع مجلد `dist` إلى Vercel/Netlify، وضبط `VITE_API_URL` لعنوان الباكند.
