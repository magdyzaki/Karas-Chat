# خطوات الرفع السريع — Karas شات

## 1. إصلاح Git والرفع إلى GitHub (من الطرفية على جهازك)

إذا ظهرت أخطاء صلاحيات:
- أغلق VS Code / Cursor بالكامل
- أغلق أي برنامج قد يستخدم المجلد (مثل OneDrive)
- افتح **PowerShell** أو **Command Prompt** كمسؤول (Run as Administrator)

ثم نفّذ:

```bash
cd D:\programs\Smart_CRM_Final_Arabic

# إزالة القفل إن وجد
del .git\index.lock 2>nul

# إضافة الملفات
git add .gitignore chat-app-new/

# commit
git commit -m "إضافة Karas شات (chat-app-new)"

# رفع للـ GitHub
git push origin main
```

---

## 2. نشر الباكند على Koyeb

1. ادخل إلى **koyeb.com** وسجّل الدخول.
2. **Create** → **Deploy** → **Create a new service**
3. **Source:** اختر **GitHub** → المستودع `Karas-Chat` (أو اسم مستودعك).
4. الفرع: **main**

### إعدادات البناء:

| الحقل | القيمة |
|--------|--------|
| Build command | `cd chat-app-new/backend && npm install` |
| Run command | `cd chat-app-new/backend && npm start` |

### متغيرات البيئة:

| Name | Value |
|------|-------|
| JWT_SECRET | `KarasChatSecret2025XYZ` (أو كلمة سر طويلة) |

5. اضغط **Deploy**
6. انتظر حتى **Running** ثم انسخ **رابط الخدمة** (مثل `https://xxx.koyeb.app`) — بدون `/` في النهاية

---

## 3. نشر الواجهة على Vercel

1. ادخل إلى **vercel.com** وسجّل الدخول (يفضل GitHub).
2. **Add New** → **Project** → اختر مستودع `Karas-Chat`.
3. **Root Directory:** اضغط Edit واكتب: `chat-app-new/frontend`
4. **Environment Variables:**

| Name | Value |
|------|-------|
| VITE_API_URL | رابط Koyeb من الخطوة 2 (مثل `https://xxx.koyeb.app`) |

5. اضغط **Deploy**
6. بعد النشر انسخ رابط المشروع — هذا رابط الشات.

---

## ملخص

| العنصر | الرابط |
|--------|--------|
| الباكند (Koyeb) | `https://xxx.koyeb.app` |
| الواجهة (Vercel) | `https://xxx.vercel.app` |

رابط Vercel النهائي = رابط Karas شات — يمكنك مشاركته مع الأصدقاء.
