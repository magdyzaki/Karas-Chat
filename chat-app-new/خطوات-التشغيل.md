# خطوات تشغيل برنامج الشات (chat-app-new)

## 1) تشغيل الباكند (السيرفر)

1. افتح **Terminal** أو **Command Prompt** أو **PowerShell**.
2. ادخل لمجلد الباكند:
   ```bash
   cd chat-app-new/backend
   ```
   *(إذا كنت داخل مجلد المشروع الرئيسي Smart_CRM_Final_Arabic)*  
   أو استخدم المسار الكامل، مثلاً:
   ```bash
   cd D:\programs\Smart_CRM_Final_Arabic\chat-app-new\backend
   ```
3. ثبّت الحزم (مرة واحدة فقط):
   ```bash
   npm install
   ```
4. شغّل السيرفر:
   ```bash
   npm run dev
   ```
5. انتظر حتى تظهر رسالة مثل: **Chat backend on 5000**  
   معناها الباكند يعمل على المنفذ **5000**.

---

## 2) تشغيل الفرونتند (الواجهة)

1. افتح **نافذة Terminal جديدة** (اترك الباكند يعمل في النافذة الأولى).
2. ادخل لمجلد الفرونتند:
   ```bash
   cd chat-app-new/frontend
   ```
   أو المسار الكامل:
   ```bash
   cd D:\programs\Smart_CRM_Final_Arabic\chat-app-new\frontend
   ```
3. ثبّت الحزم (مرة واحدة فقط):
   ```bash
   npm install
   ```
4. شغّل الواجهة:
   ```bash
   npm run dev
   ```
5. انتظر حتى يظهر رابط مثل: **http://localhost:5173**
6. افتح هذا الرابط في المتصفح.

---

## 3) الاستخدام

- من الواجهة يمكنك **إنشاء حساب** أو **تسجيل الدخول** (بريد أو رقم موبايل).
- بعد الدخول تظهر قائمة **المحادثات**، وزر **+ محادثة جديدة** لبدء محادثة فردية أو مجموعة.
- اختر محادثة لفتحها وكتابة الرسائل وإرسال صور/ملفات.

---

## ملاحظات

- **يجب أن يعمل الباكند والفرونتند معاً** حتى يعمل الشات (الباكند على 5000 والفرونتند على 5173).
- عند التشغيل المحلي **لا تحتاج** ضبط `VITE_API_URL`؛ الـ proxy في Vite يتصل تلقائياً بـ localhost:5000.
- لإيقاف التشغيل: اضغط **Ctrl + C** في نافذة الـ Terminal الخاصة بكل واحد.
