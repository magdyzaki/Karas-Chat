# تطبيق تنبيهاتي

تنبيهات مكتوبة مع تذكير **بصوت افتراضي** (قراءة النص تلقائياً). يعمل على الموبايل والكمبيوتر مع **مزامنة** عبر نفس الحساب.

## المميزات

- **كتابة التنبيه** (عنوان + نص اختياري)
- عند وقت التذكير: **إشعار مكتوب** + **قراءة النص بصوت افتراضي** (TTS)
- **مزامنة** بين الموبايل والكمبيوتر (سجّل الدخول بنفس الحساب على الجهازين)
- تطبيق ويب يمكن **تثبيته** على الموبايل (إضافة للشاشة الرئيسية) وعلى الكمبيوتر (تثبيت من المتصفح)

## متطلبات التشغيل

- **Node.js** 18 أو أحدث
- متصفح حديث (Chrome، Edge، Firefox، Safari) أو أندرويد مع Chrome
- **لا يلزم** تثبيت Visual Studio أو أدوات البناء؛ قاعدة البيانات تعمل بملف JSON (lowdb).

## التشغيل محلياً

### 1) تشغيل السيرفر (الخلفية)

```bash
cd reminders-app/backend
npm install
npm start
```

السيرفر يعمل على المنفذ **4000** (أو المتغير `PORT`).

### 2) تشغيل الواجهة (ويب)

```bash
cd reminders-app/frontend
npm install
npm run dev
```

افتح المتصفح على العنوان الذي يظهر (مثل `http://localhost:5173`).

### 3) طلب صلاحية الإشعارات

عند أول استخدام، اسمح للإشعارات في المتصفح حتى يظهر التنبيه مكتوباً ويُقرأ بصوت افتراضي.

## التثبيت على الموبايل (أندرويد) — برنامج ثابت بدون فتح المتصفح

1. تأكد أن الكمبيوتر شغال والسيرفر والواجهة يعملان، والموبايل على **نفس الواي فاي**.
2. افتح **Chrome** في الموبايل واكتب عنوان التطبيق (مثل `http://192.168.1.56:5173` حسب عنوان الـ IP عندك).
3. بعد ما الصفحة تفتح، اضغط على **النقاط الثلاث** (⋮) أعلى يمين الشاشة.
4. اختر **"إضافة إلى الشاشة الرئيسية"** أو **"تثبيت التطبيق"** أو **"Add to Home screen"**.
5. اضغط **إضافة** أو **تثبيت** في النافذة اللي تظهر.
6. هتظهر أيقونة **"تنبيهاتي"** على الشاشة الرئيسية. من دلوقتي افتح التطبيق من الأيقونة مباشرة — **بدون ما تفتح Chrome**؛ التطبيق يفتح لوحده كبرنامج مستقل (بدون شريط المتصفح).

**ملاحظة:** الأيقونة تبقى على الموبايل بشكل دائم. لما تضغط عليها التطبيق يفتح فوراً. التطبيق يشتغل لما الموبايل والكمبيوتر على نفس الواي فاي والكمبيوتر شغال. لو حابب يشتغل من أي مكان (حتى خارج البيت)، انشر السيرفر والواجهة على إنترنت (قسم "النشر على سيرفر مجاني" أدناه) ثم أضف للتطبيق الرابط الجديد من الموبايل بنفس الطريقة.

## التثبيت على الكمبيوتر

1. افتح تطبيق الويب في **Chrome** أو **Edge**.
2. في شريط العنوان يظهر أيقونة التثبيت (➕ أو "تثبيت") — اضغطها.
3. اختر "تثبيت" لفتح التطبيق كنافذة مستقلة.

---

## النشر على سيرفر مجاني (لتشغيله من أي مكان)

- **السيرفر (Backend):** يمكن نشره على [Render](https://render.com) أو [Railway](https://railway.app) (خطة مجانية).
  - أنشئ خدمة "Web Service"، اربط المستودع أو ارفع مجلد `backend`، وعيّن أمر التشغيل: `npm install && npm start`.
  - عيّن المتغير `PORT` إن لزم (غالباً تُعيّنه المنصة تلقائياً).
- **الواجهة (Frontend):** انشر مجلد `frontend` على [Vercel](https://vercel.com) أو [Netlify](https://netlify.com) (مجاناً).
  - عيّن متغير البيئة `VITE_API_URL` بعنوان السيرفر الحقيقي (مثل `https://your-backend.onrender.com`).

بعد النشر، استخدم نفس عنوان الويب على الموبايل والكمبيوتر للمزامنة.
