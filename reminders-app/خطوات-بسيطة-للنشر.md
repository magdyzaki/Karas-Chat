# خطوات بسيطة لنشر واجهة التنبيهات

---

## الأمر الواحد (نسخ والصق في PowerShell)

افتح **PowerShell** ثم نسخ والصق هذا الأمر كاملاً واضغط Enter:

```powershell
cd D:\programs\Smart_CRM_Final_Arabic\reminders-app; .\deploy-frontend.ps1
```

---

## إعدادات Vercel (بعد نجاح السكربت)

1. ادخل **vercel.com** → مشروع **reminders-frontend**
2. **Settings** → **Git** → تأكد إن الربط بمستودع **magdyzaki/reminders-frontend**
3. **Settings** → **General** → **Root Directory** = **فارغ** (امسح أي نص واتركه خالياً)
4. **Deployments** → **Redeploy**
5. انتظر دقيقة ثم افتح الموقع

---

## لو السكربت طلب صلاحيات

ربما تحتاج السماح بتشغيل السكربتات. نفّذ مرة واحدة:

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

ثم أعد تشغيل السكربت.
